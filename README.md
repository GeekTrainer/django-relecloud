# ReleCloud sample

This is a sample Django project

## Install and startup steps

```bash
# Linux/macOS/BASH
python3 -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
python manage.py runserver

# Windows
python -m venv venv
.\\venv\\scripts\\activate
pip install -r requirements.txt
python manage.py runserver
```
